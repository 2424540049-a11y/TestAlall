# 上传说明

把这个文件夹里的所有内容上传到 GitHub 仓库根目录，不要只上传这个外层文件夹本身。

必须保持以下结构：

server.js
package.json
public/
DATA/
render.yaml

Render 部署后，线上页面应引用 /app.js?v=25，并在快讯页显示沪铝/美铝新闻链接与美铝中文翻译。

本次新增的新闻阅读页文件也要上传：
- public/news-reader.html
- public/news-reader.css
- public/news-reader.js

策略下拉框显示：
- 铝稳健5%候选
- 铝防守候选

启动命令：node server.js
